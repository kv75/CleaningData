### Summary data set based on human activity recognition using smartphones data set

1. Subject
  * An identifier of the subject who performed the activity
  * Integer values from 1 to 30
2. Activity
  * Activity name
  * Strings; 6 possible values
    * LAYING
    * SITTING
    * STANDING
    * WALKING
    * WALKING_DOWNSTAIRS
    * WALKING_UPSTAIRS
3. tBodyAcc_mean_X
